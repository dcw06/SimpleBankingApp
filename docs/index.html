How to Install & Open FinWise

1

Download the DMG

Click Download (macOS) above and save FinWise.dmg to your Downloads folder.

2

Open the DMG

Go to your Downloads folder and double-click FinWise.dmg. A new window will appear showing the FinWise app.

3

Move to Applications (recommended)

Drag FinWise.app into your Applications folder to install it permanently. Or just double-click it from the DMG to run it once.

4

First Launch — Bypass Gatekeeper

Because FinWise is not signed with an Apple Developer certificate, macOS will block it on the first open. To get past this:

  1. Right-click (or Control-click) FinWise.app
  2. Select Open from the menu
  3. Click Open again in the dialog that appears

💡 You only need to do this once. After that, double-clicking works normally.

⚠️ If you see "App is damaged", open Terminal and run:
xattr -cr /Applications/FinWise.app

5

Sign In

Create an account with your email or sign in with Google, Facebook, Microsoft, GitHub, or WeChat.

6

You're in!

Explore the dashboard — run a Worker Protection Check, file a wage report, or start a financial literacy course.
